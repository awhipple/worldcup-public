# worldcup-public

Auto-generated **public** snapshot of the dojo World Cup tracker, served via
GitHub Pages. **Do not edit by hand** — every file here is produced by
`scripts/gen-static.py` in the worldcup app and overwritten on each publish.

The home server polls live data (the API key stays on the box) and pushes only
this baked, public-safe bundle (`data.json`) plus the app's static assets, so the
public never touches the server.
